# 🔐 Password Strength Checker Web App

## 📌 Overview

This is a web-based Password Strength Checker built using Flask. It evaluates the security of a password based on multiple criteria and provides suggestions to improve it.

## 🚀 Features

* Detects Weak, Medium, and Strong passwords
* Provides improvement suggestions
* Identifies common passwords
* Simple and clean UI

## 🛠️ Tech Stack

* Python
* Flask
* HTML/CSS

## 📁 Project Structure

password-strength-checker/
│── app.py
│── templates/index.html
│── static/style.css
│── requirements.txt

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/password-strength-checker.git

2. Navigate into the folder:
   cd password-strength-checker

3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   python app.py

5. Open in browser:
   http://127.0.0.1:5000
